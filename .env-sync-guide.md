# Environment Configuration Guide

## Overview
Your project uses two synchronized `.env` files for different purposes:

### 1. **SunDevilSyncUI/.env** (PRIMARY)
- **Purpose**: Server configuration for the Node.js backend
- **Used by**: `server.js` on startup
- **Contains**: Server port, session secret, blockchain keys, contract addresses, SDC rewards
- **Status**: ✅ Your main source of truth

### 2. **contracts/.env** (SECONDARY)
- **Purpose**: Hardhat configuration for smart contract deployment
- **Used by**: `hardhat.config.js` for deployments and testing
- **Contains**: Deployment keys, RPC URLs, contract addresses
- **Status**: ✅ Synchronized with SunDevilSyncUI/.env
- **Note**: Keep these in sync when updating blockchain configs

### 3. **SunDevilSyncUI/.env.example** (TEMPLATE)
- **Purpose**: Template for developers; safe to commit to git
- **Status**: ✅ Template documentation
- **Note**: Never commit actual `.env` files with secrets

## Key Variables

| Variable | Purpose | Source |
|----------|---------|--------|
| `PORT` | Server port | SunDevilSyncUI/.env only |
| `SESSION_SECRET` | Session encryption | SunDevilSyncUI/.env only |
| `BADGE_CONTRACT_ADDRESS` | Badge NFT contract | Both (sync required) |
| `SDC_TOKEN_ADDRESS` | SDC token contract | Both (sync required) |
| `PRIVATE_KEY` | Badge minting key | Both (sync required) |
| `SERVICE_WALLET_PRIVATE_KEY` | Service operations | Both (sync required) |

## Synchronization Checklist

When updating blockchain configuration:

- [ ] Update value in `SunDevilSyncUI/.env`
- [ ] Update same value in `contracts/.env`
- [ ] Verify both files have matching blockchain credentials
- [ ] Test with `npm start` in SunDevilSyncUI/
- [ ] Test with `npx hardhat test` in contracts/

## Security

⚠️ **IMPORTANT**: 
- Never commit `.env` files to git (add to `.gitignore`)
- Only commit `.env.example` as reference
- Rotate private keys regularly in production
- Use different keys for development vs. production
- Store production secrets in environment variables or secure vaults
