DEPLOYMENT FLOW:
═════════════════════════════════════════════════════════════════════════════

PHASE 1: Deploy Contracts (5 min)
    │
    ├─ cd contracts
    ├─ npm install
    ├─ npm run compile
    ├─ npm run deploy:amoy
    │
    └─ SAVE ADDRESSES:
       ├─ SunDevilBadge: 0xCc26F324F2e43F8dd1d95a25FB2DFe5659b80c34
       └─ SDCToken: 0x5d3d8d4b0D1022092e7DC3120A9798b2E58864fB


PHASE 2: Update Configuration (2 min)
    │
    ├─ Update contracts/.env
    │  ├─ BADGE_CONTRACT_ADDRESS=0xCc26F324F2e43F8dd1d95a25FB2DFe5659b80c34
    │  └─ SDC_TOKEN_ADDRESS=0x5d3d8d4b0D1022092e7DC3120A9798b2E58864fB
    │
    └─ Update SunDevilSyncUI/.env
       ├─ BADGE_CONTRACT_ADDRESS=0xCc26F324F2e43F8dd1d95a25FB2DFe5659b80c34
       └─ SDC_TOKEN_ADDRESS=0x5d3d8d4b0D1022092e7DC3120A9798b2E58864fB


PHASE 3: Start Backend (1 min)
    │
    ├─ cd SunDevilSyncUI
    ├─ npm install
    ├─ npm start
    │
    └─ Server running on http://localhost:3000


PHASE 4: Access Web App
    │
    └─ Open browser: http://localhost:3000

TROUBLESHOOTING QUICK FIX:
═════════════════════════════════════════════════════════════════════════════

Problem: "Cannot find module"
Solution: cd /Users/roystonf/Blockchain-Project/SunDevilSyncUI && npm start

Problem: "BADGE_CONTRACT_ADDRESS not configured"
Solution: Run Phase 1 deployment, copy addresses, update .env files

Problem: "Port 3000 already in use"
Solution: PORT=3001 npm start

Problem: "MetaMask network changed"
Solution: Add Polygon Amoy to MetaMask:
  RPC: https://rpc-amoy.polygon.technology/
  ChainID: 80002

Problem: "Insufficient balance"
Solution: Get MATIC tokens from https://faucet.polygon.technology/
